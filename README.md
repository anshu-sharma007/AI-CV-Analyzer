# AI CV Analyzer

AI CV Analyzer is a web application that analyzes resumes and compares candidate skills with required industry skills.

## Features

- Resume Upload
- Skill Extraction
- Profile Scoring
- Missing Skill Detection
- Recommendations Generation

## Technology Stack

- Python
- Flask
- PDFPlumber
- HTML
- CSS

## Deployment

Hosted on Render.

## Future Enhancements

- NLP Based Skill Extraction
- Machine Learning Scoring
- LinkedIn Integration
